# Pickle Ball Game using Python (Pygame)

This project is a Python micro project developed as part of the course **UGE3188 – Problem Solving and Python Programming**.  
The game is created using **Python and Pygame library** and executed using **Thonny IDE**.

---

## Project Description

Pickle Ball is a paddle-based game similar to tennis and table tennis.  
In this project, a simple Pickle Ball game is designed where two players control paddles and hit the ball.  
The game includes ball movement, paddle control, collision detection, and scoring logic.

---

## Software Requirements

- Python 3.x  
- Pygame library  
- Thonny IDE  
- Windows Operating System  

---

## How to Install Pygame

1. Open Thonny IDE  
2. Go to **Tools → Manage Packages**  
3. Search for `pygame`  
4. Click **Install**

Or use the command below in Thonny shell:

```
pip install pygame
```

---

## How to Run the Project

1. Download the project from GitHub  
   - Click **Code → Download ZIP**
   - Extract the ZIP file

2. Open **Thonny IDE**

3. Open the file `main.py`

4. Click **Run → Run current script**

5. The game window will open

---

## Controls

- **Player A (Left Paddle)**
  - W – Move Up  
  - S – Move Down  

- **Player B (Right Paddle)**
  - Up Arrow – Move Up  
  - Down Arrow – Move Down  

---

## Output

- A game window is displayed  
- Ball moves and bounces inside the window  
- Score and server information are shown  
- Game ends when winning condition is reached
